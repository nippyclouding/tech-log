관리자 창 기능
- 접속 로그 조회
- 게시글 crud
- 댓글 관리
- 카테고리 crud

access_logs 테이블에는 관리자 변경 요청, 댓글 작성/수정/삭제 요청, 관리자 API의 인증/권한 실패, 서버 오류 요청만 기록한다.
일반 공개 조회, 인증 상태 확인, 정상 로그 조회, OPTIONS/HEAD 요청은 access_logs에 기록하지 않는다.
서버 예외가 발생한 접근 로그에는 request ID와 제한된 오류 상세를 저장하여 관리자 콘솔에서 추적할 수 있게 한다.
댓글 테이블과 access_logs 테이블은 기본적으로 직접 연관관계를 맺지 않는다.
